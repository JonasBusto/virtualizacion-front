¿Como se creo el Blog?

Resumen

El blog personal consta de una estructura básica definida en secciones, que organizan la información necesaria y requerida por la materia. Se empleo HTML, CSS y JavaScript para la creación del Frontend del blog, y NodeJs como MongoDB para el Backend, donde ambas partes fueron subidas a los contenedores de Proxmox.

Configurar Proxmox

La creación del blog requiere de una configuración previa de los contenedores de Proxmox. Para ello, se instalan las herramientas necesarias en cada contenedor, tanto para FE como BE, y se sube el contenido de ambos en cada CT.
