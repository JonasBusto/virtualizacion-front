¿Como instalar apache2?

Se intentará explicar brevemente como instalar apache2 y mariaDB en un contenedor (CT). Si no dispone de un CT debe crearlo.

¿Como instalar APACHE2?

Paso 1: Dirigirse a la consola.

Para ello debe hacer click en un CT y seleccionar, entra la lista de opciones que aparecen, la opción Console.

Nota: Puede realizar este paso las veces que sea necesario.

Paso 2: Loguearse como usuario Root.

Una vez dentro de la consola, ingrese como nombre de usuario, no su DNI, sino la palabra "root". Como contraseña, ingrese la que usted definió para el acceso a la plataforma. Luego presione ENTER. Le debe aparecer como la siguiente imagen, es decir root@usuario:/#.

Paso 3: Ingresar comandos.

Una iniciado sesión como usuario root, debe escribir los siguientes comandos, en el orden que se indican:

apt update

apt install apache2

Se le preguntara si desea continuar. Presione la tecla "y".

Una vez hecho esto, debe verificar si apache2 esta instalado. Para ello escriba en la consola "apache2". Le tiene que salir información al respecto.

Para verificar la versión del apache2 escriba:

apache2 -v

Luego verifique si el apache2 está funcionando correctamente. Para ello ingrese:

systemctl status apache2

Si le aparece unas letras de color verde que dicen "Running", la instalación fue correcta. Caso contario , active apache2 ingresando:

systemctl enable apache2

¿Como instalar MariaDB?

Paso 1: Ingresar comandos.

Una iniciado sesión como usuario root, debe escribir los siguientes comandos, en el orden que se indican:

apt update

apt install mariadb-server

Se le preguntara si desea continuar. Presione la tecla "y".

Una vez hecho esto, debe verificar si mariaDB esta instalado. Para ello escriba en la consola "mariadb". Le tiene que salir información al respecto.

Para verificar la versión de mariaDB escriba:

mariadb -v

Luego verifique si la base de datos, mariadb, esta funcionando correctamente. Para ello ingrese:

systemctl status mariadb

Si le aparece unas letras de color verde que dicen "Running", la instalación fue correcta. Caso contario , active mariadb ingresando:

systemctl enable mariadb
